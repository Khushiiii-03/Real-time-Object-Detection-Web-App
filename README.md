# Real-time-Object-Detection-Web-App
An interactive web application for real-time object detection using YOLOv8 and Streamlit. The app supports image upload and live webcam detection.

Features

- Detect 80+ object classes (COCO dataset)
- Image upload detection
- Real-time webcam detection

Tech Stack

- **Python**
- **YOLOv8 (Ultralytics)**
- **Streamlit**
- **OpenCV**
- **NumPy**
- **Pillow**

How It Works

YOLOv8 performs object detection on image frames,
Streamlit provides an interactive UI and
OpenCV handles webcam and image processing.
